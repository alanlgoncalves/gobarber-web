# Project Startup

To run this project, you will need the [GoBarber API](https://github.com/alanlgoncalves/gobarber-api) running on you computer.

If you are using NPM run:

```
$ npm install

$ npm start
```

If you are using YARN run:

```
$ yarn install

$ yarn start
```
